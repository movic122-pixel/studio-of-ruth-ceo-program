# Studio of Ruth — Creative CEO Program (React + Vite + Tailwind)

This is the deploy‑ready React site for the Creative CEO Incubator curriculum page.

## Dev
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Notes
- The **Request Full Curriculum** modal embeds your Squarespace form at `https://www.studioofruth.com/creative-ceo-2` in an iframe (650px tall). Update in `src/CEOCurriculumPage.jsx` if needed.
- TailwindCSS is configured via `tailwind.config.js` and `postcss.config.js`.
- Host on **Vercel**; connect this repo and deploy.
